# Pellora

Static landing page for Pellora.

## Overview

This project is a single-page HTML site with custom CSS and lightweight JavaScript animations. It includes:

- Animated hero logo and ambient background
- Product dropdown navigation
- Vision section with three key points
- Mail contact link for inquiries

## Files

- `index.html` - Main site markup, styles, and scripts
- `pellora.logo.png` - Source logo asset
- `.gitignore` - Local files and build artifacts to ignore

## Local Preview

Run a simple static server from the project root:

```sh
python3 -m http.server 4599
```

Then open:

```text
http://localhost:4599
```

## Deployment

The site is static and can be deployed to any static hosting provider. Upload `index.html` and any required assets from the project root.
